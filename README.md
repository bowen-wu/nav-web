## 导航网站

这是一个漂亮的导航页面

#### 主要功能

1. 可以自定义 logo 

2. 使用 Google 或者 百度 进行搜索

3. 按下相应的键或者点击屏幕上的键会跳转到相应界面

4. 可以修改键盘按键所对应的网址

#### 实现效果

1. logo hover时显示【编辑】

2. logo 点击可以上传图片

3. 搜索框 hover 事件

4. 搜索是可以点击按钮，也可以按 Enter 键（默认是 Google 搜索）

5. 按钮有 hover 事件

6. 键盘按键
    - 如果有网址，跳转
    - 如果没有网址，弹出对话框

7. 点击页面上的按钮
    - 如果有网址，跳转
    - 如果没有网址，弹出对话框

8. 点击页面上按钮的 E button，弹出对话框，进行修改网址

9. 弹框确认时，可以点击确认，也可以点击 Enter 按键
    - 输入网址正确，储存到本地（localStorage）并跳转
    - 输入网址错误，输入框清空 + 提示

10. 弹框取消，重置弹框的信息 + 弹框隐藏
